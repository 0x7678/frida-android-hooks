Frida-android-hooks

Lets you hook LoadURL Method Calls in Frida ( Android )

View project on GitHub

Welcome to Firda- Hooks ( Android )

You will find different different scripts to hook various api calls in the android platform to conduct security analysis. Right now , we have two scripts :

  1. WebView loadUrl() : android.webkit.WebView
  2. getDeviceId() : android.telephony.TelephonyManager

SetUp

Setup :

1 . Load Firda -server into your android device .

$ curl -O https://build.frida.re/frida/android/arm/bin/frida-server
$ adb push frida-server /data/local/tmp/
$ adb shell "chmod 755 /data/local/tmp/frida-server"
$ adb shell "/data/local/tmp/frida-server &"

2 . Install frida on your machine . easy_install frida

3 . Change the package_name variable in the script to match the application you are instrumenting .

4 . Run : python hook_file_name_here.py

5 . Run any app that uses the instrumented methods and check the script output .

Have fun !
